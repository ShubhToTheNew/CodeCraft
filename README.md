# 🚀 CodeCraftHub

> A simple, beginner-friendly REST API for developers to track their learning journey.

CodeCraftHub is a personal learning goal tracker that helps developers organize the courses they want to learn, are currently learning, or have completed. Built with **Node.js** and **Express**, it uses a JSON file for storage—no database setup required!

## ✨ Features

- ✅ **Full CRUD Operations** — Create, Read, Update, and Delete courses
- ✅ **No Database Required** — Uses a simple JSON file for data storage
- ✅ **Auto-Generated IDs** — IDs are automatically assigned to new courses
- ✅ **Input Validation** — Ensures all required fields are present and valid
- ✅ **Status Tracking** — Track courses as "Not Started", "In Progress", or "Completed"
- ✅ **Auto-Initialization** — Creates the data file automatically on first run
- ✅ **Beginner-Friendly** — Clean code with helpful comments throughout

---

## 🛠 Tech Stack

- **Node.js** — JavaScript runtime
- **Express.js** — Web framework for building the REST API
- **JSON File** — Simple data storage (no database needed)
